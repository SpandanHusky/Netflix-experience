🎬 Netflix Login UI
A beginner-level front-end clone of the Netflix login page, built using HTML and CSS. This project focuses on understanding layout structure and improving CSS styling skills during the early stages of learning frontend development.

✨ Overview
This project represents one of my first steps into web development, where I explored how to structure a webpage and apply CSS to create a clean and familiar user interface.

🎯 Features
(1) Basic layout structure
(2) CSS styling practice
(3) Clean and simple UI
(4 )Focus on alignment and spacing

🛠️ Tech Stack
HTML
CSS

This project is a front-end clone created for educational purposes only.
